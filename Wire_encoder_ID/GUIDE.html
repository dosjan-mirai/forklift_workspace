MCP2551 DIP-8 зүү байршил: ┌────────────┐ TXD ─┤1 ● 8 ├─ Rs ──── GND (1Mbps хурд) VSS ─┤2 7 ├─ CANH ──── Encoder CAN_H VDD ─┤3 6 ├─ CANL ──── Encoder CAN_L RXD ─┤4 5 ├─ Vref (холбохгүй) └────────────┘
RXD хүчдэл хуваагч (5V → 3.3V): MCP2551 RXD (pin 4) ──┬── 10kΩ ──── PA11 (CN12 pin 10) │ 20kΩ │ GND 5V × 20/(10+20) = 3.33V ✓ STM32 PA11-д аюулгүй
Nucleo-F746ZG холбох цэгүүд: CN12 (зүүн Morpho) CN11 (баруун Morpho) ┌──┐ ┌──┐ │● │ Pin 7 → GND → MCP2551 VSS (pin 2) │● │ Pin 10 → PA11 (CAN_RX) ← хуваагчийн гаралт │● │ Pin 12 → PA12 (CAN_TX) → MCP2551 TXD (pin 1) └──┘ │● │ Pin 18 → +5V → MCP2551 VDD (pin 3) └──┘
MCP2551 RXD (pin 4) гаралт нь 5V байдаг — хуваагч резисторгүй шууд STM32-д холбовол GPIO гэмтэж болно.
Rs (pin 8) заавал GND-д холб. Холбоогүй үлдвэл standby горим — CAN дамжуулахгүй.
CAN bus-ийн хоёр үзүүрт 120Ω терминал резистор хэрэгтэй. Encoder талд ихэвчлэн суурилуулсан байдаг — datasheet шалга.

main.c — Тохиргооны утгуудыг өөрчлөх
Зөвхөн доорх 4 тодорхойлолтыг өөрчил. Бусад кодыг хүрэхгүй.
/* Core/Src/main.c — ~35-р мөр */

#define ENCODER_CURRENT_ID        0x01U   ← Encoder-ийн ОДООГИЙН CAN ID
#define ENCODER_TARGET_ID         0x02U   ← Encoder-ийг СОЛИХ CAN ID

#define ENCODER_CURRENT_BAUD_CODE WENC_BAUD_1M  ← Одоогийн baud
#define ENCODER_TARGET_BAUD_CODE  WENC_BAUD_1M  ← Солих baud

Baud код утгууд

WENC_BAUD_500K   → 500 kbps
WENC_BAUD_1M     → 1 Mbps
WENC_BAUD_250K   → 250 kbps
WENC_BAUD_125K   → 125 kbps
WENC_BAUD_100K   → 100 kbps

Жишээ: ID 1 → 3, baud хэвээр

ENCODER_CURRENT_ID        0x01U
ENCODER_TARGET_ID         0x03U
ENCODER_CURRENT_BAUD_CODE WENC_BAUD_1M
ENCODER_TARGET_BAUD_CODE  WENC_BAUD_1M
ID болон Baud-ийг нэг дор хоёуланг солих боломжтой — эхлээд ID солино, дараа нь Baud солино.
Хэрэв ID эсвэл Baud өөрчлөлт хэрэггүй бол CURRENT == TARGET гэж тавь.
Код өөрөө алхамыг алгасана.
